## 使用说明

1. 安装`Nodejs v14.10.0`版本

   > 因为使用了`node-sass`，有版本兼容问题，最好使用推荐的这个版本

3. 在源代码根目录使用`npm install`安装依赖
4. 开启本地调试：`npm run start`
5. 根据需要修改源代码
6. 打包：`npm run build:prod`
7. 打包完成后根目录会有一个`dist`文件夹，将里面的文件部署到类似`nginx`的静态服务器，访问`index.html`即可
