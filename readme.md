## 使用说明

1. 安装`Nodejs`长期使用版本
2. 在源代码根目录使用`npm install`安装依赖
3. 开启本地调试：`npm run start`
4. 根据需要修改源代码
5. 打包：`npm run build:prod`
6. 打包完成后根目录会有一个`dist`文件夹，将里面的文件部署到类似`nginx`的静态服务器，访问`index.html`即可
